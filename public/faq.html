Porter’s FAQ

Have a question? See below! We may have already answered it. And if not, feel free to ask over on Reddit and I'll add your answer here next time we push an update!

What is this?

The Chiral Cartographer is a fan-made planning tool for Death Stranding 2. It helps porters chart structures and auto-tally resources so big builds feel less like paperwork and more like a smooth delivery.

Why was it made?

Too many runs ended with scribbles on paper, half-finished phone notes, and calculator math. As a software dev (and fellow porter), I built the tool I wished existed—and I’m sharing it with the community.

Is this official?

Nope. This is a non-commercial fan project—not affiliated with Kojima Productions or Death Stranding. Just one porter practicing his craft, sharing the results, and lending a hand.

What can it do right now?

  • - Plan and queue structures
  • - Auto-tally materials per structure and in total
  • - Track roads, monorails, bridges, safe houses, zip lines, supplies by location, mines, and more!
  • - Cleaner UI for faster planning (v0.3 beta)

What’s coming next?

Strand-linked projects (share a project link with your strand so everyone can log contributions and see progress), save/load, and more visual polish based on feedback.

How can I give feedback?

Drop ideas or bug reports in the Reddit threads or Github issues page. Every porter’s perspective helps shape the Cartographer!

Does it cost anything?

No. The Cartographer is completely free! Save your resources for the roads ahead.
